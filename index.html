3D car game

Technologies: Javascript, Three.js, cannon.js, node, webpack

A three.js scene containing a car and road models I made with Blender. I used Cannon.js, a physics javascript library, to bring the car to life and allow for interactions and collisions. The car can be controlled using the arrow keys on your keyboard, and can be used to push spheres and knock over blocks.

Codenames

Technologies: HTML/CSS, Javascript, Socket.io, Node.js, Express

An online, multiplayer version of the board game Codenames. It was built using HTML/CSS, Javascript and socket.io. One player creates a new game, which others can join using the game code generated. A list of 25 words is generated and assigned to either the blue or purple team. When a player clicks a word button, an event is emitted to the whole game room, and the button will change color based on which color team the word belongs to.
